Project Report
Worked on an industrial computer vision project focusing on the visual inspection of gas cylinders in a factory in Saudi Arabia. The project involved developing a computer vision application using YOLOv5 to detect defects in gas cylinders. The application was deployed on an AI edge device (Jetson Xavier NX) and powered by ROS Noetic to manage actuators based on SMACH, a task-level architecture for execution and coordination. The system operated on a finite-state machine container provided by SMACH, responding promptly whenever a defective cylinder lacking a cap was detected, effectively diverting it from the main production line.
